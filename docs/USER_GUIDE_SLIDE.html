📊 TimeSheet Attendance Dashboard - Complete User Guide

Upload reports → Detect month → Review data → Save to cloud → Edit as needed | All steps tracked with full audit history

🚀HOW TO USE (5 Steps)

1Open: Go to https://chowdhurynumaan.github.io/iqra-timesheet-app/
2Select Month: Choose from dropdown (auto-populates available months)
3Upload Report: Click 📄 Upload Report → Select Excel file (All Reports.xls/.xlsx or CSV)
4Review Data: Check employee cards → See metrics (hours, late mins, absences) → Compare with existing data
5Save: Click ☁️ Save to Cloud → Confirm changes → Data saved with timestamps
BONUS: Click ✏️ Edit on any card to modify → Edit Tracking shows all changes made
💡 Tip: Month is auto-detected from file. Data stays in cloud, file doesn't. Firebase handles backups automatically.

⚠️COMMON ERRORS & SOLUTIONS

❌ 404 File not found
Page shows "404 - File not found"
✅ Try: Hard refresh (Ctrl+F5) or clear cache. Site uses GitHub Pages which rebuilds in ~1 min.
❌ "Cannot read properties of undefined"
Console error when uploading
✅ Try: Ensure Excel file has header row with "Date", "Name", "Hours", "Late" columns. CSV must be comma-separated.
❌ Upload button doesn't work
File selected but nothing happens
✅ Try: Check file format (must be .xls, .xlsx, or .csv). File size should be <5MB. Works on Chrome/Firefox/Safari.
❌ "No month detected"
Month dropdown empty or month not recognized
✅ Try: Ensure file has date range in format "DD-MM-YYYY ~ DD-MM-YYYY" or month name like "March 2026" in first 5 rows.
❌ Data not saving to cloud
Click save but nothing happens
✅ Try: Check internet connection. Open Console (F12) → Look for error messages. Verify Firebase initialized in browser Network tab.
